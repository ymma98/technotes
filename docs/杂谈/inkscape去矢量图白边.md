# inkscape去矢量图白边

## crop figure

* Create a rectangle covering the desired area
* select the rectangular and the figure itself together (use shift + left click to select objects)
* `Object` -> `Clip` -> `Set Clip`
* `File` -> `Document Properties ...` -> `Resize to content`
