# 一些技巧

* c++ 去除文件注释

```bash
# -fpreprocessed tells the compiler that the code is already preprocessed
# -E stops the compiler after the preprocessing stage
# -P omit line numbers** from the output.
g++ -fpreprocessed -dD -E -P test.cc
```
