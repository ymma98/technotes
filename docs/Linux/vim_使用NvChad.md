# 使用 NvChad

* [安装](https://nvchad.com/docs/quickstart/install)
* 更改主题颜色 `<leader> + th`
* NvChad 使用 Lazy 管理, 典型例子是 [`lazy_doc`](https://lazy.folke.io/spec/examples)
* Mapping
	* 查看 key mapping: `NvCheatsheet` 或 `Telescope keymaps` 
	* `<C>` -> `Ctrl`, `<leader>` -> `space`, `<A>` -> `Alt`, `S` -> `Shift`
